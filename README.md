### Setup

To set up the demo, it'll need installing Docker, `pnpm` and `poetry` globally.
E.g. in the case of macOS, `pnpm` and `poetry` can be installled via Homebrew:

```bash
brew install pnpm
brew install poetry
```

### Running

To run it, execute dev.sh in the root folder:

```bash
./dev.sh
```
